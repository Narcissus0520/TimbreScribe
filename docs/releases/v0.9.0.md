# TimbreScribe 0.9.0 prerelease

TimbreScribe 0.9.0 is the first Windows x64 prerelease of the local-first music transcription
workbench. It is intended for evaluation before the v1.0 clean-client acceptance gate closes.

## Highlights

- Import verified audio/video formats, select a range, play the source, and retain immutable media
  and transcription provenance.
- Run deterministic Mock/Test transcription or the bundled Basic Pitch ONNX CPU baseline without a
  system Python installation.
- Review and edit piano-roll/score data with undo/redo, project persistence, autosave/recovery,
  transposition, confidence filtering, and raw-versus-edited comparison.
- Export MusicXML, compressed MXL, MIDI, SVG, PNG, PDF, total scores, and individual parts.
- Use optional, explicitly gated MuScriptor and local/OpenAI-compatible assistant integrations
  without making network services or model downloads a core dependency.

## Privacy and model limitations

Core import, transcription, editing, project, and export workflows are local and require no account.
There is no telemetry by default. Audio is never sent to an assistant provider. Automatic music
transcription is probabilistic and produces an editable draft, not a guarantee of publication-ready
accuracy. MuScriptor remains experimental/non-commercial and its weights are not bundled.

## Installation and verification

Download the Windows x64 setup executable and `SHA256SUMS.txt` from this release. Verify the hash and
the Authenticode signer before installation. The per-user installer supports upgrade, `.timbrescribe`
association, and uninstall while preserving user projects and settings by default.

See `docs/USER_GUIDE.md`, `docs/TROUBLESHOOTING.md`, `docs/MODEL_LICENSES.md`, and
`THIRD_PARTY_NOTICES.md` for usage, diagnostics, licenses, and redistribution information.

## Release status

This version must be published as a GitHub prerelease. It must not be described as v1.0 or stable
until the final pristine Windows 10/11, physical-display, keyboard-only, and Narrator acceptance
matrix passes for the exact signed installer.
